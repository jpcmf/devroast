# DevRoast

**The internet's most brutal code review platform.**

DevRoast is a web application where developers submit code snippets and receive honest, sometimes hilarious feedback from the community. Think of it as code review meets comedy — where bad code gets the roasting it deserves.

## What is DevRoast?

Tired of polite code reviews? DevRoast changes the game by offering a judgment-free zone where code gets roasted with style. Whether you're writing the worst code imaginable or just curious about how others will critique it, DevRoast has you covered.

### Key Features

- **Code Submission** - Submit any code snippet in your favorite language
- **Honest Feedback** - Get real feedback without the corporate sugar-coating
- **Roast Mode** - Toggle between standard feedback and full roast mode for maximum entertainment
- **Shame Leaderboard** - See which code made it to the internet's hall of shame
- **Syntax Highlighting** - Code is displayed with beautiful syntax highlighting
- **Dark Terminal Aesthetic** - A sleek, terminal-inspired design inspired by developer culture

## How It Works

1. **Paste Your Code** - Copy and paste any code snippet into the input area
2. **Choose Mode** - Select between standard feedback or full roast mode
3. **Get Roasted** - Submit and receive instant feedback
4. **Check the Leaderboard** - See how your code ranks among the "worst" on the internet

## Design Philosophy

DevRoast embraces a dark, terminal-inspired aesthetic that appeals to developers. The interface uses:

- **JetBrains Mono** - A monospace font that feels native to code editors
- **Dark Theme** - Reduced eye strain with a sleek dark interface
- **Minimal Design** - Clean, focused layout with no distractions
- **Color-Coded Feedback** - Red for critical issues, amber for warnings, green for good code

## Built For Developers

DevRoast is designed by developers for developers. The platform celebrates the quirks of programming, bad variable names, forgotten semicolons, and all the little mistakes we all make.

No judgment. Just honest feedback and a good laugh.

## Getting Started

To use DevRoast:

1. Visit the website
2. Paste your code in the input area
3. Toggle roast mode if you want extra spice
4. Click "Start the Roast" and await your verdict
5. Share your results or learn from the shame leaderboard

## Community Leaderboard

The leaderboard showcases the most roasted code on the internet. It's a gallery of programming mishaps, creative solutions gone wrong, and moments of brilliance mixed with embarrassment.

Check out the leaderboard to:
- See what NOT to do
- Learn from others' mistakes
- Find inspiration for your next roast

## About

DevRoast is built with modern web technologies and a passion for honest code feedback. It's a place where developers can laugh at themselves and grow together.
